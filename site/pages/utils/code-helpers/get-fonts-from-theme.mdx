# getFontsFromTheme()

## Syntax

<Code>{'getFontsFromTheme(defaultToken, customProp)(props)'}</Code>

## Overview

Retrieve a font token from the current theme. Font tokens are CSS values to be to change typography including values for: font-weight, letter-spacing, font-family and line-height.

## Parameters

<LegacyBlock $marginTop='sizing040'>
  <Prop name='defaultToken' type='ThemeToken'>
    Is the name of the font token to be retrieved. If provided, it is used as an object key on the font section of the theme to get the token value.
  </Prop>
  <Prop name='customProp' type="Exclude<keyof Props, 'theme'>">
    Is the name of the component prop in which a font token name can be proivded by the end-user of the component. If provided, the font to be retrieved will come from the component prop.
  </Prop>
  <Prop name='props' type='Props'>
    Is the props of the component including the theme. If the function is used inside a styled component template literal, or a [css block](/utils/css), [Emotion](https://emotion.sh/docs/) will call the function first with the component props passed in.
  </Prop>
</LegacyBlock>

## Return value

Returns a font token if valid attributes are received.

Returns an empty string if invalid attributes or no attributes are provided.

## Example

<CodeFromFile language="text" path="examples/utils/get-fonts-from-theme.tsx"/>

In this example the `Paragraph` component will use the font-weight value from `fontWeight010` found in the theme if the `Paragraph` is used with no props:

<Code>{'<Paragraph />'}</Code>

However, it will include the value from `fontWeight040` if the `Paragraph` is used with the `$fontWeight` prop and the value passed in is `fontWeight040`:

<Code>{'<Paragraph $fontWeight="fontWeight040" />'}</Code>
