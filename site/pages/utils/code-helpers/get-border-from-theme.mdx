# getBorderFromTheme()

## Syntax

<Code>{'getBorderFromTheme(defaultToken, customProp)(props)'}</Code>

## Overview

Retrieve a border token from the current theme. Border tokens are CSS values to control border styles.

## Parameters

<Block $marginTop='sizing040'>
  <Prop name='defaultToken' type='ThemeToken'>
    Is the name of the border token to be retrieved. If provided, it is used as an object key on the border section of the theme to get the token value.
  </Prop>
  <Prop name='customProp' type="Exclude<keyof Props, 'theme'>">
    Is the name of the component prop in which a border token name can be proivded by the end-user of the component. If provided, the border to be retrieved will come from the component prop.
  </Prop>
  <Prop name='props' type='Props'>
    Is the props of the component including the theme. If the function is used inside a styled component template literal, or a [css block](/utils/css), [Emotion](https://emotion.sh/docs/) will call the function first with the component props passed in.
  </Prop>
</Block>

## Return value

Returns a border token if valid attributes are received.

Returns an empty string if invalid attributes or no attributes are provided.

## Example

<CodeFromFile language="text" path="examples/utils/get-border-from-theme.tsx"/>

In this example the `Paragraph` component will include the border value from `borderWidth010` found in the theme if the `Paragraph` is used with no props:

<Code>{'<Paragraph />'}</Code>

However, it will include value from `borderWidth030` if the `Paragraph` is used with the `$border` prop and the value passed in is `borderWidth030`:

<Code>{'<Paragraph $border="borderWidth030" />'}</Code>
