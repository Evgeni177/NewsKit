# Button

## Overview

A styled button element with customisable size.

## Props

Buttons have a number of props to facilitate a variety of uses. Any prop valid on an Button HTML element as described in [here](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#Attributes), is also valid on the Button component. 
Custom props are detailed below:

<LegacyBlock marginTop="sizing040">
  <Prop name="$size" type="ButtonSize" enum={ButtonSize} default={ButtonSize.Small}>
    Set the size of the button. Possible values: Small, Medium and Large. Default styling: Small.
  </Prop>
    <Prop name="$stylePreset" type="string" default="interactive010">
    Set the style presets of the button.
  </Prop>
</LegacyBlock>

### Styling

The button component is block level element with a native HTML button with custom styling.

### Accessibility

The button is a fully accessible with the native HTML button used.

#### Colour

Text and background colours must have a 3:1 minimum contrast ratio.
Hover states should have clear contrast change or should include a non colour designator.
