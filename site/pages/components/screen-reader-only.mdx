# ScreenReaderOnly

## Overview

The screen reader only component contains description of an element, which the screen reader can use.

### Usage

The `ScreenReaderOnly` component requires a paragraph passed as a child.

## Props

<LegacyBlock $marginTop="sizing040">
  <Prop name="bylineData" type="array" required>
    An array of byline data to populate the byline authors. Byline data objects
    must contain an author string, and can contain optional href and title
    strings. See playground for example.
  </Prop>
</LegacyBlock>
