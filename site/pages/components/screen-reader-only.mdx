# Screen Reader Only

## Overview

The screen reader only component contains description of an element, which the screen reader can read.

## Props

<LegacyBlock $marginTop="sizing040">
  <Prop name="children" type="string" required>
    The description of an element is passed as the child of the component.
  </Prop>
  <Prop name="id" type="string" required>
    The CSS id required for the `aria-described-by` of the described element. 
  </Prop>
</LegacyBlock>
