# Optimizely Web

## Overview

The Optimizely Web component is a non-visual component for the injection of the Optimizely Web 3rd party script and custom script for enabling Optimizely based on cookie.

## Prerequisites

To utilise this component you will need to have the relevant Optimizely Web project in the Optimizely platform and Optimizely script CDN URL.

## Props

<LegacyBlock marginTop="sizing040">
  <Prop required name="optimizelyCdn" type="string" required>
    Optimizely script CDN URL.
  </Prop>
  <Prop name="reactHelmet" type="react-helmet Helmet Component">
    By default the Optimizely Web component will inject its scripts by rendering script
    tags. However, if your project is using [React
    Helmet](https://www.npmjs.com/package/react-helmet) you can pass the Helmet
    component in and the scripts will render using that component.
  </Prop>
