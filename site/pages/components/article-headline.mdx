# Article Headline
The Article Headline is a simple component that comprises two element: a Kicker (text) and Heading (text).  

The kicker is optional.  
They can be both be rendered as different HTML elements.  

## Overview

### Usage

No special requirements.

### Theming

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="children" type="string" required>
    The Heading text of the Article Headline is passed as the child of the component.
  </Prop>
  <Prop name="kickerText" type="string">
    The optional Kicker content, which appears before the Heading.
  </Prop>
  <Prop name="renderHeadingAs" type="string" default="h1">
    The type of tag you want to render the Heading as.
    Must be one of 'h1','h2','h3','h4','h5','h6','span'.
  </Prop>
  <Prop name="renderKickerAs" type="string" default="span">
    The type of tag you want to render the Kicker as.
    Must be one of 'h1','h2','h3','h4','h5','h6','span'.
  </Prop>
  <Prop name="headingStylePreset" type="string">
    The name of the style preset you want to overwrite the default style with.
    The value must be the name of another valid style preset.
  </Prop>
  <Prop name="headingMqTypePreset" type="object">
    Object containing the type presets for the Heading breakpoints. They will overwrite the default ones.
    Example object keys:

      xs: 'code010',
      sm: 'heading060',
      md: 'code020',
      lg: 'heading080',
    
    The key's value must be a type presets.
  </Prop>
  <Prop name="kickerMqTypePreset" type="object">
    Object containing the type presets for the Kicker breakpoints. They will overwrite the default ones.
    Example object keys: 

      xs: 'code010',
      sm: 'heading060',
      md: 'heading060',
      lg: 'heading060',

    The key's value must be a type presets.
  </Prop>
  <Prop name="kickerMqInlineMarginPreset" type="object">
    Object containing the inline margin presets for the Kicker breakpoints. They will overwrite the default ones.
    Example object keys: 

      xs: 'spaceInline060',
      sm: 'spaceInline060',
      md: 'spaceInline070',
      lg: 'spaceInline080',

    The key's value must be a spacing presets.
  </Prop>
  <Prop name="kickerStylePreset" type="string">
    The name of the style preset you want to overwrite the default style with.
    The value must be the name of another valid style preset.
  </Prop>
</LegacyBlock>
