# Share Bar

## Overview

The share bar uses [icons](/foundations/icons) to allow users to share.

### Usage

A share bar can be used anywhere a user would like to easily share something with social media.

## Props

<LegacyBlock $marginTop='sizing040'>
  <Prop name='leftIcons' type='array'>
    An array of social items that start from the left.
  </Prop>
  <Prop name='rightIcons' type='array'>
    An array of social items that start from the right.
  </Prop>
  <Prop name='leftLabel' type='string'>
    A label to appear before the left items.
  </Prop>
  <Prop name='rightLabel' type='string'>
    A label to appear before the right items.
  </Prop>
</LegacyBlock>

## Rationale

### Styling

Share bars give the user the ability to share articles.

The articles can be shared via social media or saved locally for later use.

### Accessibility

#### Keyboard navigation
The share bar uses an unordered list to follow accessibility standards.

If a user focuses on an element and presses enter it will do the same action as if the user clicked on it.
