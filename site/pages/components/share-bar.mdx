# Share Bar

## Overview

The share bar uses [icons](/foundations/icons) to allow users to share.

### Usage

A share bar can be used anywhere a user would like to easily share something with social media.

## Props

<LegacyBlock $marginTop="sizing040">
  <Prop name="label" type="string">
    If provided with text, adds a label and populates it with the string.
  </Prop>
  <Prop name="vertical" type="boolean" defaut="false">
    If provided, changes the orientation of the share bar to vertical.
  </Prop>
  <Prop name="labelStylePreset" type="string" default="shareBarLabel">
    If provided, overwrites the style preset for the label.
  </Prop>
  <Prop name="labelSpacePreset" type="MarginPresetKeys" default="spaceInline040">
    If provided, overwrites the spacing preset for the label.
  </Prop>
  <Prop name="labelTypePreset" type="string" default="shareBarLabel">
    If provided, overwrites the type preset for the label.
  </Prop>
  <Prop name="itemSpacePreset" type="MarginPresetKeys" default="sizing020">
    If provided, overwrites the spacing preset for the stack.
  </Prop>
</LegacyBlock>

## Rationale

### Styling

Share bars give the user the ability to share articles.

The articles can be shared via social media or saved locally for later use.

### Accessibility

#### Keyboard navigation

The share bar uses an unordered list to follow accessibility standards.

If a user focuses on an element and presses enter it will do the same action as if the user clicked on it.
