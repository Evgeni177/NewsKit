# Image

## Overview

Simple component for adding images.
The image component provides a `<figure>` and an `<img>` element inside it to place the image in the dom.

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="src" type="string">
    Image file path that is used directly in the 'src' attribute of the internal
    'img' element.
  </Prop>
  <Prop name="alt" type="string">
    Image description that is used directly in the 'alt' attribute of the
    internal 'img' element.
  </Prop>
  <Prop name="width" type="string" default="100%">
    Used in combination with `height` to calculate the aspect ratio for the
    image. Generally supply the width px value if know, if not supply horizontal
    aspect ratio. Both width and aspectRatio should be supplied in a consistent
    manner.
  </Prop>
  <Prop name="maxWidth" type="string">
    see https://developer.mozilla.org/en-US/docs/Web/CSS/max-width
  </Prop>
  <Prop name="height" type="string" default="auto">
    Used in combination with `width` to calculate the aspect ratio for the
    image. Generally supply the height px value if know, if not supply vertical
    aspect ratio. Both width and aspectRatio should be supplied in a consistent
    manner.
  </Prop>
  <Prop name="maxHeight" type="string">
    see https://developer.mozilla.org/en-US/docs/Web/CSS/max-height
  </Prop>
  <Prop name="loadingAspectRatio" type="string">
    This is a string interpolation of the aspect ratio of the loading container
    of the image. This is also used to determine the height or width of the
    image if only one dimension is provided.
  </Prop>
  <Prop name="hideLoadingIcon" type="boolean" default={false}>
    Hide the loading icon whilst the image is loading.
  </Prop>
  <Prop name="captionText" type="string">
    If provided, supplies the caption component. captionText is the text of the caption.
  </Prop>
  <Prop name="creditText" type="string">
    If provided, supplies the credit text of the caption component.
  </Prop>
  <Prop name="overrides" type="object">
    If provided, overrides the respective presets for the component and provided elements.
    <Prop name="stylePreset" type="MQ<string>">
      If provided, this overrides the style preset applied to the image.
    </Prop>
    <Prop name="caption" type="object" >
      If provided, this overrides the styling of the caption component.
      <Prop name="typographyPreset" type="MQ<string>">
        If provided, this overrides the typography preset of the caption component.
      </Prop>
      <Prop name="stylePreset" type="MQ<string>">
        If provided, this overrides the style preset of the caption component.
      </Prop>
      <Prop name="spaceStack" type="MQ<string>">
        If provided, this overrides the spacing between the caption and the
        credit.
      </Prop>
      <Prop name="spaceInset" type="MQ<string>">
       If provided, this overrides the padding of the Caption. This override should only be used
       with CaptionInset. Otherwise, spaceInset040 or spaceInset050  is used depending on the `size` prop.
      </Prop>
      <Prop name="credit" type="object" >
        If provided, this overrides the styling of the credit portion of the caption component.
          <Prop name="typographyPreset" type="MQ<string>">
            If provided, this overrides the typography preset of the credit text.
          </Prop>
          <Prop name="stylePreset" type="MQ<string>">
            If provided, this overrides the style preset of the credit text.
          </Prop>
      </Prop>
    </Prop>
    Refer to the defaults below for the object structure:
    <CodeFromDefaultPresets componentName="image" />
  </Prop>
</LegacyBlock>

### Styling

The image component is block level element and it wraps its boundaries around the internal img element and the dimenstions of the image file provided. Change or limit the width of the component by styling the returned wrapper.

Placeholder icon can be overridden in the theme with a different icon if needed.

### Accessibility

Provide an alternative text description for the image via its alt prop. This will allow screen readers and other text based browsers to understand the image.

#### Colour

Text and background colours must have a 3:1 minimum contrast ratio.
Hover states should have clear contrast change or should include a non colour designator.
Placeholder icon colour is taken from `inkNonEssential` token
Placeholder background colour is taken from `skeleton010` token
