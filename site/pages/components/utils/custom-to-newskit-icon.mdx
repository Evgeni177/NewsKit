# customToNewsKitIcon()

## Syntax

<Code>{'customToNewsKitIcon(displayName, CustomIcon, overrides)'}</Code>

## Overview

Transform SVG markup into a Newskit icon that works well with theme tokens and `getFromTheme` util functions.

## Parameters

<LegacyBlock marginTop="sizing040">
  <Prop name="displayName" type="string" required>
    The display name of the icon. It is required an icon name to start either
    with <b>`IconFilled`</b> or <b>`IconOutlined`</b> prefix, depending on the
    imported icon type. <br />
    E.g: If importing an <b>MenuBook</b> icon, the final icon display name should
    be <b>`IconFilledMenuBook`</b> or <b>`IconOutlinedMenuBook`</b>
  </Prop>
  <Prop name="CustomIcon" type="React.ComponentType<SvgProps>" required>
    NewsKit Svg component with SVG passed to it's children, props should be
    spread on the root Svg component, see example below.
  </Prop>
  <Prop name="overrides" type='SvgProps["overrides"]'>
    Default <a href="/components/icons#props">override props</a> passed to
    the transformed icon. Overrides can still be passed to the icon to overwrite
    these values at render time.
  </Prop>
</LegacyBlock>

## Return value

Returns a NewsKit icon.
