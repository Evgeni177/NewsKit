# Unordered List

## Overview

### Usage

An unordered list can be used to produce a list of related content prefixed with no icons as default, this can be optionally overridden with a custom icon.

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="children" type="Array<React.ReactNode>">
    An array of react nodes to populate the list components.
  </Prop>
  <Prop name="listItemMarker" type="React.ComponentType<{}>">
    Optional icon component to override the list style.
  </Prop>
</LegacyBlock>
