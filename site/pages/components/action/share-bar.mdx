# Share Bar

## Overview

The share bar uses [icon buttons](/components/action/button#playground-for-icon-button) to allow users to share content.

### Usage

A share bar can be used anywhere a user would like to easily share something with social media.

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="label" type="string">
    If provided with text, adds a label and populates it with the string.
  </Prop>
  <Prop name="vertical" type="boolean" defaut={false}>
    If provided, changes the orientation of the share bar to vertical. Note that
    the 'spaceInline' will still apply between the vertically orientated label
    and icons.
  </Prop>
  <Prop name="overrides" type="object">
    If provided, overrides the respective presets for the component and provided
    elements.
    <Prop name="label" type="object">
      <Prop name="stylePreset" type="MQ<string>">
        If provided, this overrides the style preset applied to the passed
        label.
      </Prop>
      <Prop name="typographyPreset" type="MQ<string>">
        If provided, this overrides the typography preset applied to the passed
        label.
      </Prop>
      <Prop name="spaceInline" type="MQ<string>">
        If provided, this overrides the horizontal space between the items and
        label.
      </Prop>
      <Prop name="spaceInset" type="MQ<string>">
        If provided, this overrides the padding of the share bar.
      </Prop>
    </Prop>
    Refer to the defaults below for the object structure:
    <CodeFromDefaultPresets componentName="shareBar" />
  </Prop>
</LegacyBlock>

## Rationale

### Styling

Share bars give the user the ability to share articles.

The articles can be shared via social media or saved locally for later use.

### Accessibility

In order to make the share bar accessible for screen readers you should pass a role="region" and
an aria-label to the share bar.

_NOTE: if you have more than one share bar on your page - all share bar components (or their containers)
should have `role` set to region and also an unique `aria-label`._

#### Keyboard navigation

The share bar uses an unordered list to follow accessibility standards.

If a user focuses on an element and presses enter it will do the same action as if the user clicked on it.
