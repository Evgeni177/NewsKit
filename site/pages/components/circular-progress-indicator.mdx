# Circular progress indicator

## Overview

Circular progress indicators show the progression of a system operation such as downloading, uploading, processing, etc. in a visual way.

## Behaviour

Circular progress indicators display progress by animating the indicator along the length of the track. The circular indicator fill the circular track with colour, as it moves from 0 to 360 degrees over a quicker duration and repeats the animation until the action has been completed.

## Usage

Circular progress indicators display progress by animating an indicator along an invisible circular track in a clockwise direction. They can be applied to buttons, inputs or other elements.

Don't show more than one circular progress indicator as the user is likely to become overwhelmed! If more than one piece of content is processing, stick to a large for the whole screen and consider skeleton loading or linear progress indicators.

### Accessibility

When applying a progress indicator then the aria-busy state should be applied so that users know content is being updated.
