# Link

## Overview

A simple styled anchor element.

### Usage

A link requires a href property to be passed in.

### Theming

The default link component uses the following style tokens, which can be overridden:

| Theme Token | Color Primitive | Color Value                               |
| ----------- | --------------- | ----------------------------------------- |
| linkText    | inkLink          | <TextBoxSwatch themeColor="inkLink"/>      |
| linkVisited | inkLinkVisited     | <TextBoxSwatch themeColor="inkLinkVisited"/> |
| linkHover   | inkLink         | <TextBoxSwatch themeColor="inkLink"/>     |
| linkActive  | inkLink        | <TextBoxSwatch themeColor="inkLink"/>    |

## Props

Links have a number of props to facilitate a variety of uses. Any prop valid on an Anchor HTML element, is also valid on the Link component. Custom and required props are detailed below:

<LegacyBlock marginTop="sizing040">
  <Prop name="children" type="string" required>
    The content of the link is passed as the child of the component.
  </Prop>
  <Prop name="href" type="string" required>
    Passed to the href attribute of the rendered anchor element. The provided
    URL or fragment identifier will be loaded when the link is clicked.
  </Prop>
  <Prop name="font" type="TypePresetKeys">
    If provided, uses that type preset. Without this, none is set and the
    underlying anchor element will inherit its parent font properties.
  </Prop>
  <Prop name="color" type="ColorKeys">
    If provided, overrides the text color of the link including psuedo
    selectors. Overriding individual pseudo selectors can be done via a theme.
  </Prop>
  <Prop name="noUnderline" type="boolean">
    If provided, determines if the link will be underlined.
  </Prop>
</LegacyBlock>
