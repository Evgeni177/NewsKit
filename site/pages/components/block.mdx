# Block

## Overview

A Block component is a simple div component that only applies margin using the fixed spacing preset set in the theme.
For more information on the spacing preset, please refer to page [here](/foundations/spacing).

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="margin" type="MarginPresetKeys">
    This is a margin prop that only accepts stack and inline preset tokens defined in the spacing theme.
    see https://developer.mozilla.org/en-US/docs/Web/CSS/margin
  </Prop>
  <Prop name="as" type="string">
    This is a string name of html element that the block can be converted to.
    Block defaults as a div element, but it can be converted to a span by `as="span"` prop.
  </Prop>
</LegacyBlock>
