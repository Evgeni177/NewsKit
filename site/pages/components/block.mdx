# Block

## Overview

The Block is a simple container component that can take margin, padding and style presets. 
For more information on the spacing preset, please refer to page [here](/foundations/spacing).

## Props

<LegacyBlock marginTop="sizing040">
  <Prop name="as" type="string" default="div">
    This is a string name of the html element that the block can be converted to.
    Block defaults as a div element, but it can be converted to a `span` by using this prop.
  </Prop>

  <Prop name="overrides" type="object">
    If provided, overrides the respective presets for the component and provided elements.
    Here are the overrides that the Block accepts:
    <Prop name="paddingPreset" type="PaddingPresetKeys">
      If provided, applies padding on the Block.
    </Prop>
    <Prop name="stylePreset" type="string">
      If provided, applies style on the Block.
    </Prop>
    <Prop name="marginPreset" type="MarginPresetKeys">
      If provided, applies margin on the Block.
    </Prop>
    This component does not have any default preset.
  </Prop>
</LegacyBlock>
